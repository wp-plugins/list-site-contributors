List site contributors (and authors). 

== Description ==

List site contributors allows you list your site's contributors. (http://www.mallsop.com/plugins)
Just add the shortcode [listsitecontributors] to a page on your blog. 
Optionally you can use the User Photo Plugin (http://wordpress.org/extend/plugins/user-photo/) for images,  
otherwise Avatars will be used.

Hat tip to Brian at watershedstudio.com for keeping me in the world of Wordpress.

== Installation ==

1. Extract and upload the directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= What does this do? =

Shows a listing of active contributors (and authors) with images with an A-Z directory and last name search.

= What if I do not have the User Photo Plugin? =

It will use the User Photo Plugin image, if installed. Otherwise the Wordpress Avatar image will be used. 

= Where are the options? =

The only option now is yes/no for showing the url for each author/contributor.
